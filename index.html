GitHub Veröffentlichung

Um diese App auf GitHub Pages zu veröffentlichen:

  1. Erstellen Sie ein Repository auf GitHub
  2. Laden Sie alle Dateien hoch
  3. Aktivieren Sie GitHub Pages in den Repository-Einstellungen
  4. Die App ist erreichbar unter: https://[username].github.io/[repository-name]

Für die Entwicklung: Stellen Sie sicher, dass alle Pfade zu den CSS- und JS-Dateien korrekt sind.
